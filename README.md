# Bash-JSON
Simple JSON parser for bash.

## Requirements
- GNU Bash 4.4

## Installation
1. `git clone https://github.com/HuaiShaoChang/Bash-JSON.git`
2. `cd Bash-JSON`
3. `. install.sh`

## Usage
- `cat <json_file> | bj`
